    • On-Demand Instances
    • Pay for compute capacity by the second or hour with no long-term commitments.
    • Best for short-term, unpredictable workloads or testing environments.
    • No upfront payment and no long-term contract.
    Reserved Instances (RIs)
    • Commit to a 1- or 3-year term for significant discounts (up to 72% off On-Demand pricing).
    • Types:
      • Standard RIs: Biggest discount but less flexibility.
      • Convertible RIs: Allow changing instance attributes but offer lower discounts.
      • Scheduled RIs: Reserve capacity for specific recurring schedules.
    Savings Plans
    • Flexible pricing model offering lower costs based on a committed spend per hour for 1 or 3 years.
    • Types:
      • Compute Savings Plan: Applies to any EC2 instance, regardless of region, instance family, OS, or tenancy.
      • EC2 Instance Savings Plan: Applies to a specific instance family and region for better discounts.
    Spot Instances
    • Up to 90% cheaper than On-Demand instances.
    • Ideal for fault-tolerant and flexible workloads like batch processing, big data, and CI/CD.
    • Instances can be terminated by AWS when capacity is needed.
    Dedicated Hosts
    • Physical servers dedicated to your use, allowing compliance with licensing and regulatory requirements.
    • Best for enterprises with strict compliance needs.
    Dedicated Instances
    • Run on hardware that is physically dedicated to a single customer, but without the management features of Dedicated Hosts.
    • Useful for security or regulatory requirements.
    Capacity Reservations
    • Reserve capacity in a specific Availability Zone without any long-term commitment.
    • Can be combined with RIs or Savings Plans for cost savings.
    EC2 Instance Purchase Options - Exam MCQs

    1. Which EC2 purchase option provides the highest discount for long-term workloads?

    • A) On-Demand
    • B) Reserved Instances
    • C) Spot Instances
    • D) Dedicated Hosts

    Answer: B) Reserved Instances

    Explanation: Reserved Instances offer up to 72% discount over On-Demand pricing when committing to a 1- or 3-year term.

    2. What is the main advantage of using Spot Instances?

    • A) Provides consistent performance
    • B) Reduces cost by up to 90%
    • C) Guaranteed availability
    • D) Always available in all regions

    Answer: B) Reduces cost by up to 90%

    Explanation: Spot Instances are ideal for workloads that are fault-tolerant and can handle interruptions.

    3. Which EC2 purchase option is best for an application that runs only for a few hours every week at a predictable time?

    • A) On-Demand
    • B) Spot Instances
    • C) Scheduled Reserved Instances
    • D) Dedicated Instances

    Answer: C) Scheduled Reserved Instances

    Explanation: Scheduled Reserved Instances allow reserving capacity for specific time slots in a week, making them cost-effective.

    4. What happens when AWS needs Spot Instance capacity back?

    • A) AWS provides a 2-minute warning before terminating the instance
    • B) AWS moves the instance to another region
    • C) AWS automatically converts it into an On-Demand instance
    • D) AWS pauses the instance and resumes later

    Answer: A) AWS provides a 2-minute warning before terminating the instance

    5. What is the key difference between Dedicated Hosts and Dedicated Instances?

    • A) Dedicated Hosts allow using existing software licenses
    • B) Dedicated Instances provide more flexibility in instance types
    • C) Dedicated Hosts are more cost-effective
    • D) Dedicated Instances allow multi-tenancy

    Answer: A) Dedicated Hosts allow using existing software licenses

    6. How can a company ensure that a critical EC2 instance always has available capacity in a specific AZ?

    • A) Purchase Spot Instances
    • B) Use a Capacity Reservation
    • C) Use a Compute Savings Plan
    • D) Use Reserved Instances

    Answer: B) Use a Capacity Reservation

    7. What is the best EC2 purchase option for a disaster recovery environment that must always be available?

    • A) Spot Instances
    • B) Reserved Instances
    • C) On-Demand
    • D) Savings Plans

    Answer: B) Reserved Instances

    8. Which EC2 instance type is best for unpredictable workloads that need flexibility?

    • A) On-Demand
    • B) Spot Instances
    • C) Reserved Instances
    • D) Dedicated Hosts

    Answer: A) On-Demand

    9. What happens if a company purchases a Compute Savings Plan?

    • A) They must use the same instance type
    • B) The discount applies to any EC2 usage
    • C) It only applies to a specific region
    • D) The savings are refundable

    Answer: B) The discount applies to any EC2 usage

    10. Which EC2 pricing model provides the most flexibility without long-term commitment?

    • A) On-Demand
    • B) Spot Instances
    • C) Reserved Instances
    • D) Dedicated Hosts

    Answer: A) On-Demand
