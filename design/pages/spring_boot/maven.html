What is maven?
  • Maven is a powerful project management tool.
  • It uses the pom.xml file, where POM stands for Project Object Model.
  • It manages project builds, dependencies, and documentation.
  • When a Maven command is executed, it looks for the pom.xml file.
  • In Maven, we specify what to do.
  • In Ant, we specify both what to do and how to do it.
Understanding of pom.xml
    schema:
    xmlns and xsi:schemaLocation define the XML namespace and schema for Maven POM files. These attributes are standard for all Maven pom.xml files. 
    <project xmlns="http://maven.apache.org/POM/4.0.0" 
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    Model Version:
    This specifies the version of the POM model 
    <modelVersion>4.0.0</modelVersion>
    Uniquely identifies the project: 
    <groupId>com.example</groupId>
<artifactId>spring-boot-app</artifactId>
<version>1.0.0</version>
<packaging>jar</packaging>
<name>spring-boot-app</name>
<description>Spring Boot Application</description>
    Parent POM (Spring Boot Starter Parent) 
    <!-- Parent Spring Boot Starter -->
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>3.2.2</version>
    <relativePath/> 
</parent>

    The parent section allows you to inherit configurations from another POM.
    Here, it's inheriting from the Spring Boot starter parent, which provides default configurations for Spring Boot applications.

    Properties 
        <properties>
        <java.version>17</java.version>
    </properties>

    The properties section allows you to define project-specific properties.
    java.version specifies the Java version to be used for this project (Java 17 here)

    <dependencies>
    <!-- Spring Boot Starter Web -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
